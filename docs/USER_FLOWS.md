# User Flows

## 1. Onboarding & Partner Connection
- **Step 1**: User A signs up via Email/Password or Social Auth.
- **Step 2**: User A completes their profile (Name, Anniversary, Photo).
- **Step 3**: User A generates a unique invitation link or code.
- **Step 4**: User B receives the link, signs up, and enters the code.
- **Step 5**: The "Couple" entity is created, and both users are redirected to the shared dashboard.

## 2. Shared Memory Creation
- **Step 1**: Either user clicks "Add Memory" on the dashboard.
- **Step 2**: User uploads a photo and adds a title/description/date.
- **Step 3**: On save, the memory is added to the timeline.
- **Step 4**: The partner receives a push notification: "New memory added by [Partner Name]! ❤️"

## 3. Weekly Relationship Check-in
- **Step 1**: Dashboard displays a "Check-in" prompt every Sunday.
- **Step 2**: Both users answer 3 simple questions privately (e.g., "How did you feel this week?").
- **Step 3**: Once both complete it, the results are revealed to both.
- **Step 4**: Tips are generated based on the responses (e.g., "You both mentioned feeling busy—plan a quiet night in!").

## 4. Shared Goals & Milestones
- **Step 1**: Users navigate to the "Goals" tab.
- **Step 2**: Add a goal (e.g., "Save for Italy trip").
- **Step 3**: Update progress or mark as completed.
- **Step 4**: Confetti animation on completion!
